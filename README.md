# Expense Manager CLI (Python)

A simple command-line expense tracker built with Python.

## Features

* Add expenses with amount, category, description, and date
* View all saved expenses
* Delete specific expenses
* Show total spending summary
* Data stored in JSON format
* Menu-based CLI interface

## How It Works

The program stores all expenses in a JSON file (`data.json`).
Users interact with the application through a command-line menu.

## Menu Options

1. Add Expense
2. View Expenses
3. Delete Expense
4. Summary
5. Exit

## Technologies Used

* Python
* JSON file storage
* Git / GitHub
